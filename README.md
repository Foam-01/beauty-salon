# 💇‍♀️ Beauty Salon Website

A modern landing page for a beauty salon that showcases services, promotions, and customer reviews.  
Designed with a clean UI and responsive layout for a better user experience.

## 🔧 Tech Stack

- Frontend: React.js + Tailwind CSS
- Framework: Next.js
- Icons: Lucide React
- Deployment: Netlify
- Tools: GitHub, VS Code

## 🚀 Features

- 💇‍♀️ Salon Services Showcase
- ⭐ Customer Reviews Section
- 🎁 Promotion / Special Offers
- 📞 Contact Information
- 📱 Responsive Design (Mobile Friendly)
